# Branding

## Current package branding
- Product name: **MFA Sidecar**
- Temporary logo/icon: **Authelia logo**
- Description style: **Authelia-based MFA sidecar for YunoHost domains and paths**

## Rationale
This package is a heavy YunoHost-specific packaging/integration layer around Authelia, not a fork of Authelia itself.
Using the upstream Authelia logo for now is truthful and low-friction while the package identity matures.

## Source
Logo asset sourced from Authelia's official branding guide/repository assets:
- branding guide: `docs/content/reference/guides/branding.md`
- current packaged asset: `package-base/assets/logo.png`

The upstream branding guide explicitly allows reasonable size/format/layout adjustments while asking that usage not imply endorsement or affiliation.

## UI positioning
- This package should be treated as an **admin-side/operator tool**.
- It should **not** appear as a normal end-user dashboard tile/app.
- The logo is for package/admin presentation, not for ordinary user launch UX.
